# Asistente Construccion

Shell base de la aplicacion preparada con React, TypeScript y Vite.

## Scripts

- `npm run dev`: inicia el entorno local.
- `npm run typecheck`: ejecuta el verificador de tipos.
- `npm run build`: compila la aplicacion para produccion.
